Privacy Policy for LeetTracker Chrome Extension

LeetTracker is a Chrome Extension that helps users track their LeetCode submission history and problem-solving progress. This extension is designed for personal, local use only.

What data does the extension access?

  • It reads your own LeetCode submission data using LeetCode's internal APIs.
  • It may optionally fetch problem descriptions to enrich local tracking.
  • It accesses data on leetcode.com and communicates with the companion web app on leet-tracker-log.vercel.app (or localhost during development).

What data is stored or shared?

  • All submission data is stored locally in your browser using Chrome's storage.local API and IndexedDB.
  • No submission data, personal data, or problem content is sent to any external server.
  • The LeetTracker web app runs entirely in your browser and does not transmit or persist data beyond your local machine.
  • Analytics data (see below) is sent to PostHog for usage tracking and error monitoring to improve the extension.

Analytics and Usage Tracking

LeetTracker uses PostHog, a privacy-focused analytics platform, to collect usage data. This helps us understand how the extension is used and identify bugs or performance issues.

What analytics data is collected?

  • LeetCode username and user ID: Used for debugging and correlating events
  • Usage events: Extension sessions, sync operations, feature usage
  • Performance metrics: Sync duration, enrichment times, backfill progress
  • Error reports: JavaScript errors, API failures, lock timeouts
  • Extension metadata: Version number, environment (dev/prod)

What is NOT collected?

  • Submission code or problem solutions
  • Problem descriptions or content
  • Personal notes or annotations

Data Retention

Analytics events are retained by PostHog for 90 days and then automatically deleted. No long-term storage of analytics data occurs.

How to Opt Out

While the extension does not currently have a built-in toggle to disable analytics, you can block analytics tracking using any of these methods:

  • Chrome Extension Settings: Right-click the LeetTracker icon → "Manage Extension" → scroll to "Site access" → click "Add" under "Blocked sites" → enter https://us.i.posthog.com/*
  • Content Blockers: Use browser extensions that block analytics (e.g., uBlock Origin with us.i.posthog.com filter)
  • Firewall/DNS: Configure your browser or network to block PostHog domains

Blocking analytics will not affect the extension's core functionality. The extension will continue to sync and track your submissions normally.

We respect your privacy. A future version may include an opt-out toggle in the extension settings.

Third-party data usage

LeetTracker accesses LeetCode’s GraphQL API and may fetch problem content. These are copyrighted by LeetCode. You are responsible for ensuring that your use of this data complies with LeetCode’s Terms of Service and Privacy Policy.
