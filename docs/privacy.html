Privacy Policy for LeetTracker Chrome Extension

LeetTracker is a Chrome Extension that helps users track their LeetCode submission history and problem-solving progress. This extension is designed for personal, local use only.

What data does the extension access?

  • It reads your own LeetCode submission data using LeetCode's internal APIs.
  • It may optionally fetch problem descriptions to enrich local tracking.
  • It accesses data on leetcode.com and communicates with the companion web app on leet-tracker-log.vercel.app (or localhost during development).

What data is stored or shared?

  • All data is stored locally in your browser using Chrome’s storage.local API and IndexedDB.
  • No submission data, personal data, or problem content is sent to any external server.
  • The LeetTracker web app runs entirely in your browser and does not transmit or persist data beyond your local machine.

Third-party data usage

LeetTracker accesses LeetCode’s GraphQL API and may fetch problem content. These are copyrighted by LeetCode. You are responsible for ensuring that your use of this data complies with LeetCode’s Terms of Service and Privacy Policy.

Disclaimer

This extension is for personal, non-commercial use. It is not affiliated with or endorsed by LeetCode. Users must not publish, distribute, or expose LeetCode problem content retrieved via this extension.
